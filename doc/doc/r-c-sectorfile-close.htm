Reference: Classes: SectorFile

close

The close function closes the Sector file.

int close
(
);
Parameters

None.

Return Value

On success, the function returns 0. If error occurs, a negative value is returned.

Error Name Error Code Comment
     
Description

The close function releases the Sector file and makes it ready to open another file. To release the pointer/handle itself, use Sector::releaseSectorFile.
