Reference: Classes: SectorFile

upload

The upload function uploads a local file to the Sector system.

int upload
(
   cconst char* localpath,
   const bool& cont = false
);
Parameters
localpath
[in] the name of the local file to be uploaded.
cont
[in] if the destination file already exists, this flag specifies if the file is uploaded from the existing offset or overwrite the existing file.
Return Value

On success, the function returns 1. If error occurs, a negative value is returned.

Error Name Error Code Comment
     
Description

The upload function does not change the current write offset position.
