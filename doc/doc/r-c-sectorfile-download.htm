Reference: Classes: SectorFile

download

The download function downloads a file from Sector to local disk.

int download
(
   const char* localpath,
   const bool& cont = false
);
Parameters
localpath
[in] the local file name to store the downloaded file.
cont
[in] if the local file already exists, the function may continue to download from the offset of the local file size (assuming it is a broken download from previous try) or overwrite the local file.
Return Value

On success, the function returns 1. If error occurs, a negative value is returned.

Error Name Error Code Comment
     
Description

The download function does not change the current read offset position.
