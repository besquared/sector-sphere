Reference: Classes

SInput

The SInput class is a the parameter for the input of User Defined Functions (UDFs).

Class Members
Member Type Description
m_pcUnit char* The data segment to be processed
m_iRows int number of rows/records in this segment
m_pllIndex int64_t* An array of 64-integer that notes the offset of each records in the input segment
m_pcParam char* the parameter passed here by the Sphere application in run() or run_mr()
m_pSize int size of the parameter buffer
Class Interfaces

There is no public interface for this class.
