Reference: Tools

download

The download tool downloads one or more files from the Sector system.

download FILE/DIR LOCALPATH
Parameters
FILE/DIR
The source file or directory in the Sector system .
LOCALPATH
The local directory to store the downloaded files.
Description

The download tool downloads a file or a whole directory to the local file system. The local directory must exist.

Wildcard characters * and ? are supported to specify the source files or directories.
