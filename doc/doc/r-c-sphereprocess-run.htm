Reference: Classes: SphereProcess

run

The run function starts the execution of a Sphere process.

int run
(
   const SphereStream& input,
   SphereStream& output,
   const string& op,
   const int& rows,
   const char* param = NULL,
   const int& size = 0
);
Parameters
input
[in] the input stream for the Sphere processing
output
[out] the output stream that stores the processing result files
op
[in] the name of the user defined function
rows
[in] how many rows should Sphere process each time, -1: the complete data segment, 0: the file, n>1: n rows
param
[in] parameter for the user defined function
size
[in] size of the parameter buffer
Return Value

On success, the function returns 0. If error occurs, a negative value is returned.
