move

The move function moves a file/directory from one directory to another, and/or rename the current one to the specified destination name.

int move
(
   const string& oldpath,
   const string& newpath
);
Parameters
oldpath
[in] the name of a file or directory to be moved and/or renamed .
newpath
[in] the name of destination file or path.
Return Value

On success, this function return 1. If error occurs, a negative value is returned.

Error Name Error Code Comment
     
Description

Suppose oldpath is "src_dir/src_name" and newpath is "dst_dir/dst_name". odlpath and "dst_dir" must exist, otherwise E_NOEXIST error will return. If "dst_name" exist and it is file rather than a directory, E_EXIST error will return. If "dst_name" is an existing directory, the file/directory "src_name" will be moved to "dst_dir/dst_name/src_name". If "dst_name" does not exist, the file/directory "src_name" will be moved to "dst_dir" and renamed to "dst_dir/dst_name".
