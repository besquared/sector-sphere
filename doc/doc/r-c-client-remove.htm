remove

The remove function deletes a file or a directory from the system.

int remove
(
   const string& path
);
Parameters
path
[in] the name of a file or directory to be deleted.
Return Value

On succes, the function returns 1. If error occurs, a negative value is returned.

Error Name Error Code Comment
     
Description

Currently Sector does not provide any mechanism to recover deleted files.
