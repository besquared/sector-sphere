Reference: Classes: Sector

createSectorFile

The createSectorFile function creates a new Sector file handler.

SectorFile* createSectorFile
(
);


Parameters
None.
Return Value

On success, the function returns the pointer to a new Sector file. If error occurs, NULL is returned.

Error Name Error Code Comment
     
Description

Note that the Sector file must be created by this way and be released by releaseSectorFile. The methods new and delete are disabled for SectorFile.
