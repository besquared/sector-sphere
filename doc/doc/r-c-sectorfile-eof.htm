eof

The eof function checks if the read position is already at the end of the file.

bool eof
(
);
Parameters

None.

Return Value

The function returns true if end of file is reached, otherwise false is reached. No error value can be returned.

Description

This function is usually used together with read() to check if there is more data to read.
