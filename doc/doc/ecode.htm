Reference: Error Codes

Sector/Sphere Error Codes List

Sector returns negative value when error occurs. The value indicates the type of error.

Error Name Error Code Comment
E_UNKNOWN -1 undefined error
E_PERMISSION -1001 no permission for requested operation
E_EXIST -1002 file or directory already exist
E_NOEXIST -1003 file or directory does not exist
E_BUSY -1004 file is busy
E_LOCALFILE -1005 local file error
E_NOEMPTY -1006 directory is not empty
E_NOTDIR -1007 directory does not exist or not a directory
E_SECURITY -2000 security check failed
E_NOCERT -2001 no certificate found
E_ACCOUNT -2002 user does not exist
E_PASSWORD -2003 incorrect password
E_ACL -2004 visit from illegal IP address
E_INITCTX -2005 failed to initialize SSL CTX
E_NOSECSERV -2006 security server is down or cannot connect to it
E_EXPIRED -2007 connection time out due to no activity
E_CONNECTION -3000 cannot set up connection or connection is broken
E_RESOURCE -4000 no available resources
E_NODISK -4001 no enough disk space
E_TIMEDOUT -5000 operation timeout
E_INVALID -6000 invalid parameter
E_SUPPORT -6001 operation not supported
E_BUCKET -7001 Sphere bucket failure
