Reference: Tools

cp

The cp tool makes a copy of an existing file/director within the Sector DFS.

cp SRC_PATH DST_PATH
Parameters
SRC_PATH
The file/directory to be copied.
DST_PATH
The target path .
Description

Suppose SRC_PATH is "src_dir/src_name" and DST_PATH is "dst_dir/dst_name". SRC_PATH and dst_dir must exist, otherwise "file not exist" error will return. If "dst_name" exist and it is a file rather than a directory, "file already exist" error will return. If "dst_name" is an existing directory, the file/directory "src_name" will be copied to "dst_dir/dst_name/src_name". If "dst_name" does not exist, the file/directory "src_name" will be copied to "dst_dir" and renamed to "dst_dir/dst_name".

Wildcard characters * and ? are supported to specify the source files or directories.

The cp tool only works for files within the Sector DFS. To copy a file in to or out of the Sector DFS, use the upload or download tool.
