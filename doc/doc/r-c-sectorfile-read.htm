Reference: Classes: SectorFile

read

The read function reads a block of data from the opened file.

int64_t read
(
   char* buf,
   const int64_t& size
);
Parameters
buf
[out] the pointer to buffer to accept the data.
size
[in] size of data to read.
Return Value

On success, the function returns the actual number of bytes that has been read. If error occurs, a negative value is returned.

Error Name Error Code Comment
     
Description

The function will read data from the current read offset position, which is 0 initially and can be reset by seekg. The offset can checked by tellg.
