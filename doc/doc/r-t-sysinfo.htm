Reference: Tools

sysinfo

The sysinfo tool retrieves the detailed informtaion of the sector system.

sysinfo
Parameters
None.
Description

Below is an example of the output of the sysinfo tool. The first part shows the system level information, including system start time, available disk, total file size, total number of files, and total number of slave nodes. The second part shows the cluster level information. Only the top level of clusters is shown. The information includes total number of nodes, available disk space, total file size, total size of data moved into the cluster, and total number of data moved out of the rack. The third part shows the node level information. It shows, for each node, the IP address, start time, available disk, total size of files, memory used by Sector, CPU time used by Sector (since the start time), total size of inbound data from network, and total size of outbound data.
