Welcome to the Sector/Sphere documentation.

Sector/Sphere is a system for distributed data storage, distribution, and processing. The system works on clusters of commodity computers. Sector provides client tools to access data stored in the system and API for the development of distributed data processing applications.

Visit http://sector.sf.net for most recent news.
