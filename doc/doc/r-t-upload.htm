Reference: Tools

upload

The upload tool uploads one file or the whole directory to the Sector system.

upload SRC_FILE/DIR DST_DIR
Parameters
SRC_FILE/DIR
The local file or directory to be uploaded.
DST_DIR
The destination directory in the Sector system.
Description

The upload tool uploads one file or a whole local directory "SRC_FILE/DIR" to the Sector system. The uploaded files will be put in the DST_DIR directory. The destination directory DST_DIR must exist.
