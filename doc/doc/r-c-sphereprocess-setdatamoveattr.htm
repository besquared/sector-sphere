setDataMoveAttr

The setDataMoveAttr function specifies if the input data can be read from other nodes.

void setDataMoveAttr
(
   bool move
);
Parameters
move
[in] whether input data reading from other nodes is allowed
Return Value

This function does not return any value.

Description

In certain situation, by reading data only from its location node may speed up the process.
