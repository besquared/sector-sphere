resizeIdxBuf

The resizeIdxBuf resizes the result data record index bufferof the Sphere User Defined Functions, in case that it is not large enough.

int resizeIdxBuf
(
   const int64_t& newsize,
);
Parameters
newsize
[in] The size of the new index buffer.
Return Value

On success, the function returns the new size of the index buffer.

Error Name Error Code Comment
     
Description

This function can be used to allocate a large index buffer if the system provided buffer is not large enough. The buffer size unit is int64_t. The bucket buffer is also resized to the same new size.
