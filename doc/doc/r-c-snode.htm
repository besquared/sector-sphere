Reference: Classes

SNode

The SNode class is a descriptor of a Sector file or directory. This is like the inode structure in Unix file system.

Class Members
Member Type Description
m_strName string The name of the file or directory
m_bIsDir bool If this SNode is a directory
m_sLocation set<Address> Set of addresses that the file or its replica is located
m_llTimeStamp int64_t Last modified time
m_llSize int64_t Size of the file
Class Interfaces

There is no public interface for this class.
