Reference: Classes

SFile

The SFile class is used as one of the parameters in Sphere User Defined Functions (UDFs). It is used to pass file related information between Sphere server and UDF.

Class Members
Member Type Description
m_strHomeDir string The base directory of the Sector file storage on the current slave node.
m_strLibDir string The directory that stores user uploaded library files in the current session.
m_strTempDir string The directory that UDF can write temporary data to.
m_sstrFiles set<string> A set of user modified files. Sector uses this list to update the file system.
Class Interfaces

There is no public interface for this class.
