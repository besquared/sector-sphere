resizeResBuf

The resizeResBuf resizes the result buffer of the Sphere User Defined Functions, in case that it is not large enough.

int resizeResBuf
(
   const int64_t& newsize,
);
Parameters
newsize
[in] The size of the new buffer.
Return Value

On success, the function returns the new size of the index buffer.

Error Name Error Code Comment
     
Description

This function can be used to allocate a larger buffer when the system provided buffer is not large enough.
