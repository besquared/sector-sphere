seekp

The seekp function reset the current write offset position.

int seekp
(
   int64_t off,
   int pos = SF_POS::BEG
);
Parameters
off
[in] The new relative offset.
pos
[in] The base position to recalculate the offset. The available options are defined in structrure SF_POS:
Option Comment
SF_POS::BEG the head of the file
SF_POS::CUR current position
SF_POS::END the end of the file
Return Value

On success, the function returns 1. If error occurs, a negative value is returned.

Error Name Error Code Comment
     
Description

The new offset will be set to (pos + off). If pos is BEG, the offset is calculated from the 0. If pos is CUR, the offset is based on the current position. Finally, if pos is END, the offset is based at the end of the file. The final position must be greater than 0.
