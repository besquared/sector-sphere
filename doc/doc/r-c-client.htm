Reference: Classes

Client (alias: Sector)

The Client class contains methods that maintain the connection with the Sector system and global operations on the file system.

Class Members

There is no public member variables for this class.

Class Interfaces
Method Fuctionality
close close and release the current client
init initialize the Sector client
list retrieve the list of files or sub-directories of the specified directory
login login a user
logout logout a user
move move/rename a file or a directory
remove delete a file or a directory
rmr delete a file or a directory recursively
stat query the information about a file or a directory
sysinfo query the Sector system information
