Sector/Sphere Classes

This section describes in detail the Sector/Sphere classes, including:

Class Description
Address network address of a Sector node
Sector the Sector client
SectorFile A Sector file handler
SFile The file parameter of a User Defined Function (UDF)
SNode A file or directory in the Sector file system
SInput The input parameter of a UDF
SOutput The output parameter of a UDF
SphereProcess A Sphere process
SphereResult The result of a data segment in a Sphere processing
SphereStream A Sphere stream, the operand of a Sphere process
SysStat Sector system information
