login

The login function logins a user with a password.

int login
(
   const string& username,
   const string& password,
   const char* cert = NULL
);
Parameters
username
[in] Name of the user to login .
password
[in] The user's password in clear text (not encrypted).
cert
[in] path to the master certificate "master_node.cert", default (NULL) means the certificate can be found in the current directory.
Return Value

On success, the function returns a unique session key assigned by the Master. If error occurs, a negative value is returned:
