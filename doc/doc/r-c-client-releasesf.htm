Reference: Classes: Sector

releaseSectorFile

The releaseSectorFile function releases a Sphere process handler.

int releaseSectorFile
(
   SectorFile* sf
);


Parameters
sf
[in/out] the pointer to the SectorFile to be released.
Return Value

On success, 0 is returned. Otherwise a negative value is returned to indicate the error (NULL pointer, not a Sector file, etc.).

Error Name Error Code Comment
     
Description

This function is used to release a SectorFile handler created by createSectorFile.
