utime

The utime function updates teh time stamp of a Sector file.

int utime
(
   cconst char* path,
   const int64_t & ts
);
Parameters
path
[in] the name of the Sector file to be updated.
ts
[in] new time stampe.
Return Value

On success, the function returns 0. If error occurs, a negative value is returned.

Error Name Error Code Comment
     
Description
