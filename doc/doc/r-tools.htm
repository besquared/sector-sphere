Sector Tools

This section describes in detail the Sector file system client tools. These tools are located in ./codeblue2/client/tools.

The client needs to know the address of the master server (IP/domain name:PORT), the user name/the password, and the location of the master certificate. These information can be configured in the configuration file of ./client/client.conf. If the file does not exist, or you leave some of the items out (e.g., password), the client tools will prompt for user input.

Method Fuctionality
cp make a copy of a file/directory.
download download a list of files to local disk.
ls list the content of the specified directory.
mkdir create a new directory.
mv rename a file/directory or move it to another directoy.
rm delete a file or a directory.
stat query the information of a file or a directory.
stream run Sphere operations from command lines.
sysinfo retrieve the system information.
upload upload a local file to the Sector system.
