Reference: Classes: SectorFile

write

The write function writes a block of data to the opened file.

int64_t write
(
   const char* buf,
   const int64_t& size
);
Parameters
buf
[in] the pointer to the buffer of data to write to the Sector file.
size
[in] size of the data to write.
Return Value

On success, the function returns the actual number of bytes that has been written. If error occurs, a negative value is returned.

Error Name Error Code Comment
     
Description

This function writes a block of data from the current write offset position, which is 0 initially and can be reset by seekp. The write offset can be retrieved by tellp.
